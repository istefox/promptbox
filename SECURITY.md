# Security Policy

## Supported versions

The plugin has no published release yet. Once released, only the latest version on the Obsidian community store is supported.

## Reporting a vulnerability

Report vulnerabilities privately via [GitHub Security Advisories](https://github.com/istefox/promptbox/security/advisories/new). Do not open a public issue for security problems.

You can expect an initial response within 7 days.

## Scope notes

Promptbox MVP is fully local: no server, no network calls, no telemetry. Vault-level security (encryption, access control) is explicitly out of scope (`docs/spec.md` NG-5); issues in that area belong to Obsidian itself or your sync tooling.
