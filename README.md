# Promptbox

Obsidian plugin: a local-first library of AI prompts with categorization, search, and quick reuse. Prompts are plain Markdown notes with YAML frontmatter, so the library stays readable and portable even without the plugin.

## Status

Bootstrap, under active development. The full planning package (specification, ADRs, tiered implementation plan) lives in [`docs/`](docs/); no plugin code has been written yet.
